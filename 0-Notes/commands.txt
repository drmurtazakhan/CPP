To save .cpp files from local PC to github
Add, Commit, and Push. Type following commands one by one
git add .
git commit -m "Updated all files"
git push origin main

OR type a single commanmd to do all tasks at once
git add . && git commit -m "Bulk update all files" && git push origin main

To force Pull (get) from github to local CPP

git pull origin main
git fetch origin
git reset --hard origin/main

Exclude Files in VS Code 
Press Ctrl + , (comma) to open your Settings.

In the search bar at the top, type "exclude".

Look for the section called Files: Exclude.

Click Add Pattern.

To hide everything that is not a .cpp file, you would usually have to list the things you don't want to see. Add these patterns one by one:

**/*.exe (Hides all Windows programs)
----------------------

**/*.o (Hides all object files)

**/*.out (Hides Mac/Linux programs)
